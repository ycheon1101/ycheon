# projects

Projects

alt = LiDAR UDA figure
Robust Perception
2025
LiDAR Unsupervised Domain Adaptation for Semantic Segmentation
Implemented and extended UniMix for adverse-weather 3D LiDAR semantic segmentation, introducing confidence-aware spatial mixing and controllable target-region selection to improve domain adaptation performance by +3.6% mIoU over baseline.

alt = COLMAP-Free 3DGS and Depth Map Analysis figure
3D Vision
2025
COLMAP-Free 3DGS and Depth Map Analysis
Re-implemented COLMAP-free 3D Gaussian Splatting and benchmarked depth estimation models (MiDaS, ZoeDepth, Depth Anything) on synthetic video sequences, finding ZoeDepth achieved the best performance across PSNR, SSIM, LPIPS.

alt = Graphcut-based segmentation refinement figure
Computer Graphics
2024
Graphcut-Based Post-Processing for Semantic Segmentation
Proposed an entropy-based Graph Cut post-processing method for semantic segmentation refinement, achieving +2.64% mIoU over CRF-RNN.

alt = Visual question answering pipeline figure
VQA
2024
Visual Question Answering with Zero-shot Models
Developed a zero-shot visual question answering pipeline using Grounding DINO and SAM, achieving 80.2% accuracy in zero-shot evaluation and outperforming a traditional VGG + LSTM-style baseline on unseen data.

alt = ML optimization comparison figure
ML Optimization
2024
Evaluating Optimization Algorithm for Machine Learning
Conducted a comparative analysis of various optimization algorithms on semantic segmentation and image classification tasks, evaluating their generalization performance.

alt = Multi-GPU training from scratch figure

2023
Multi-GPU Training from Scratch
Implemented PyTorch Distributed Data Parallel (DDP) from scratch across 2 GPUs for an 8-layer MLP, resolving memory constraints while maintaining 96% accuracy on MNIST and improving training speed by ~1.7%.
